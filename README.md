💳 Credit Card Financial Dashboard – Power BI
An interactive dashboard analyzing customer behavior, transactions, and revenue trends from 10,000+ credit card records. This Power BI project empowers stakeholders with data-driven insights into customer segmentation, revenue drivers, and card performance.

📊 Dashboard Overview


1. Credit Card Customer Report
KPIs:
🏦 Total Revenue: 55M
💰 Total Interest: 7.8M
📈 Total Income: 576M
⭐ Customer Satisfaction Score (CSS): 3.19


Visuals Include:
Revenue trends by Gender and Time
Revenue by:
Age Group
Marital Status
Income Group
Education Level
Number of Dependents
Top 5 Performing States
Customer Job-wise Revenue Breakdown


3. Credit Card Transaction Report
KPIs:
🏦 Total Revenue: 55M
💰 Total Interest: 7.8M
💳 Transaction Amount: 45M
🔁 Transaction Count: 656K


Visuals Include:
Quarterly Revenue & Transaction Trends
Revenue by Card Type (Silver, Blue, Gold, Platinum)
Revenue by Use Type: Swipe, Chip, Online
Revenue by Expenditure Type: Bills, Entertainment, Travel, etc.
Customer Acquisition Cost per Card Category


📦 Dataset
10,000+ records
Fields: Customer demographics, Transaction data, Card type, Channel used, Interest, Income, etc.
Used for segmentation and time-series analysis


🛠️ Built With
Power BI Desktop
Power Query – for cleaning and transforming large datasets
DAX – for creating custom KPIs and calculated metrics
Slicers – for dynamic filtering across multiple dimensions
